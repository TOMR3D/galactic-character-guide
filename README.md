# Galactic Character Guide 🌌
 
A Star Wars character analyser from the Jedi Archives on Coruscant, built using SWAPI.

---

## About

Take a look at characters from a galaxy far, far away.
Browse heroes, villains, and species from the Star Wars universe, all powered by SWAPI.

This is a **fan-made educational project**.

---

## Features 

- Browse Star Wars characters across the galaxy
- View characters details including species, homeworld, and films
- Clean SwiftUI layout for easy navigation

---

## Data Source

Character data is provided by **[SWAPI - Star Wars API](https://swapi.info)**
